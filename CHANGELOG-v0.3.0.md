## What's New in v0.3.0

### New Features
- **BIOS fallback for mod aircraft** -- When the palette opens for an aircraft without DCS-BIOS JSON (e.g. modded Super Hornet FA-18E), it suggests the most similar stock aircraft and offers to use its BIOS controls instead. The override is saved in settings and cleared when you change aircraft.
- **Inline state for binary switches** -- Simple toggle switches (OBOGS, ALT BARO/RDR, etc.) now show their current state directly in search results without needing to open a submenu.
- **Position labels for binary switches** -- 40+ binary switches now show proper named labels (RDR/BARO, SAFE/ARM, NORM/EMER, DAY/NIGHT, etc.) instead of generic ON/OFF.

### Bug Fixes
- Submenu header now hides the BIOS identifier when "Show DCS-BIOS Identifiers" is disabled in settings
- Removed duplicate description in submenu (already shown in search bar above)
